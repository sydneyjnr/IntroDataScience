# Data Science Summary

## Python
Programming language. Easy to read. Used for data science, web, AI.
- Get it: python.org
- Check: `python --version`
- Types: strings, numbers, booleans
- Structures: lists, dicts, tuples, sets
- Loops: if/else, for, while
- Functions: reusable code
- Modules: math, random, etc
- Errors: try/except

## Git
Track file changes. Save work history. Collaborate with team.
- Setup: name and email
- `git init` - new repo
- `git add .` - save files
- `git commit` - save with message
- `git push` - upload to GitHub
- `git pull` - download changes
- Branches: create/switch/merge

## UV
Fast Python package manager. Installs packages super quick.
- Install stuff: `uv add package-name`
- Run code: `uv run python file.py`
- Start project: `uv init my-project`
- Sync: `uv sync`
- Way faster than old pip

## Marimo
Python notebook. Better than Jupyter.
- Files are .py (pure Python)
- Auto-updates cells (reactive)
- Works with Git
- `uv run marimo edit file.py`
- Export to HTML

## Polars
Fast data tool. Like Excel but way better and faster.
- Read CSV/JSON: `pl.read_csv()`
- Pick columns: `.select()`
- Filter rows: `.filter()`
- Count/sum: `.group_by().agg()`
- 5-10x faster than pandas

## Plotly
Make beautiful charts. Interactive and clickable.
- Scatter plots: `px.scatter()`
- Line charts: `px.line()`
- Bar charts: `px.bar()`
- Histograms, box plots, pie charts
- Customize colors, labels, themes
- Export to HTML

## VS Code
Code editor. Free. Has tools built in.
- Write code here
- Built-in terminal
- Git buttons in sidebar
- See debug output
- Extensions for everything

## Markdown
Simple text formatting. No buttons needed.
- `#` = heading
- `**text**` = bold
- `*text*` = italic
- `- item` = bullets
- `[link](url)` = links
- Good for docs and README

## Data Science
Turn data into useful info.
- Collect data

- Find patterns
- Make predictions
- Tell the story

Used everywhere: business, healthcare, tech, finance.
