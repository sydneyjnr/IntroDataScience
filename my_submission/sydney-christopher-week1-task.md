# Week 1 – Tools Overview and Understanding

## Introduction
In Week 1, I learned about the main tools and concepts that will be used throughout the data science training. These tools focus on writing clean Python code, managing projects and dependencies, handling and analyzing data efficiently, visualizing insights, and documenting work clearly using Markdown.

---

## Python
Python is the primary programming language used in this training. It is popular in data science due to its simple syntax, readability, and strong selection of libraries. Python allows for easy writing of clear, maintainable code while supporting complex data analysis tasks.

---

## UV (Package and Project Management)
UV is a modern tool for managing Python projects and dependencies. It simplifies project setup, installs dependencies, and manages environments. Compared to traditional tools like pip, UV is faster and helps ensure reproducible environments, which is important for teamwork and long-term projects.

---

## Git and Version Control
Git is used for version control, which tracks changes in code over time. It makes collaboration easier by letting multiple people work on the same project safely. Git also allows developers to revert to earlier versions of code and keeps a clear history of project changes.

---

## VS Code
Visual Studio Code (VS Code) is the main code editor used in this training. It offers features like syntax highlighting, extensions, integrated terminals, and Git support. These features help boost productivity and make working with Python, Markdown, and data science tools more efficient.

---

## Marimo Notebooks
Marimo notebooks are interactive notebooks made for reproducible data science workflows. They keep code, outputs, and explanations connected and reduce issues with hidden execution states. This setup makes analysis clearer and easier to maintain compared to traditional notebooks.

---

## Polars (Data Wrangling)
Polars is a high-performance DataFrame library used for data manipulation. It is built for speed, memory efficiency, and parallel execution. Polars uses an expressive and type-safe API that encourages clean data transformations through method chaining and expressions.

__Key operations in Polars include:__
- Reading data from formats such as CSV, JSON, and Parquet  
- Selecting and filtering columns and rows  
- Sorting data and modifying or creating new columns  
- Grouping and aggregating data  
- Joining multiple datasets  
- Handling missing values  
- Using lazy evaluation for large datasets to boost performance  

_Polars promotes good data habits by being immutable, efficient, and optimized for modern data workloads._

---

## Plotly (Data Visualization)
Plotly is a visualization library used to create interactive charts and graphs. It allows users to explore data visually and communicate insights effectively. Interactive visualizations are particularly useful for understanding patterns and presenting results clearly.

---

## Data Science Fundamentals
Data science involves collecting, cleaning, analyzing, and interpreting data to generate insights. It combines programming, statistics, and critical thinking. This training emphasizes reproducibility, performance, and clear communication as key principles of effective data science work.

---

## Markdown
Markdown is a lightweight markup language for writing structured documentation. It allows users to create readable documents using simple syntax. Markdown is commonly used for README files, reports, and documentation, making it an essential tool for communicating technical work clearly.
